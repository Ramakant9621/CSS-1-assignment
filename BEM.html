Block (.paragraph): The main container for each paragraph.

element (.paragraph__title): The heading inside each paragraph container.

Modifier (.paragraph--important and .paragraph--highlighted):

These classes modify the style of the paragraph block. The paragraph--important class indicates an important paragraph, while the paragraph--highlighted class indicates a highlighted paragraph.
